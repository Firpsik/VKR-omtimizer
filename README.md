# АСОП-Маркет

Веб-приложение для оптимизации цен на маркетплейсах (Wildberries, Ozon,
Яндекс Маркет). По данным продаж считает оптимальную цену, объём и
ожидаемую прибыль по каждой площадке и схеме фулфилмента.

Выпускная квалификационная работа.

## Что внутри

База данных хранит каталог товаров, историю продаж и тарифные
правила маркетплейсов. Тарифы организованы по схеме Class Table
Inheritance: общая часть в `mp.tariff_rule`, спецификация каждой
площадки — в подтаблицах `tariff_rule_wb / tariff_rule_ozon /
tariff_rule_ym`. Юнит-экономика считается SQL-функциями, объединяется
во view `mp.v_unit_economics`.

Спрос моделируется линейно `q = a − b·s`; коэффициенты оцениваются
МНК-регрессией по `mp.sales_history` (предварительно отфильтрованной
от промо-наблюдений, stockout и выбросов 2.5σ). Оптимальная цена
`s* = ½(a/b + C/m)`, объём `q* = a − b·s*`, прибыль
`П* = (m·s* − C) · q*`.

Каждый зарегистрированный пользователь видит только свои товары и
продажи. Демо-аккаунт открыт для просмотра без регистрации.

## Стек

| Слой | Технология |
|---|---|
| База | PostgreSQL 16 (Yandex Managed Service) |
| Бэкенд | FastAPI + SQLAlchemy + psycopg |
| Расчёты | NumPy + SciPy |
| Деплой | Docker, Yandex Container Registry, Serverless Containers |
| BI | Yandex DataLens поверх `mp.v_dashboard` |

## Структура

```
.
├── data/                     CSV-источники: канонические категории, тарифы, логистика
├── sql/migrations/           SQL-миграции, применяются по алфавиту
├── src/
│   ├── api/                  FastAPI + шаблоны
│   ├── core/                 ядро: оценка спроса, оптимизация
│   ├── etl/                  загрузчики и адаптеры CSV-отчётов МП
│   ├── auth.py               пароли, сессионные cookies
│   └── db.py                 подключение к PostgreSQL
├── tests/                    сценарные тесты
├── certs/root.crt            корневой сертификат Yandex Cloud
├── Dockerfile                образ для Serverless Containers
└── redeploy.ps1              пересборка и деплой
```
